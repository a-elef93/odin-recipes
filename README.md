# Odin Recipes

This project is part of The Odin Project curriculum.

The goal of this project is to build a basic recipe website using only HTML.  
The website will include a main index page with links to several recipe pages.

## Skills Demonstrated

By completing this project, I will have demonstrated:

- Basic HTML structure
- Creating links between pages
- Working with lists
- Adding images
- Organizing files and folders
- Using Git and GitHub for version control

## Project Structure

- `index.html` → Main homepage
- `recipes/` → Contains individual recipe pages

## Future Improvements

In the future, this project will be styled with CSS to improve the appearance and layout of the website.
